# ⚽ Copa da Sustentabilidade - Agrinho 2026

## 📋 Informações do Projeto
- **Tema:** Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente
- **Subcategoria:** Subcategoria 3 - Programação Front-End (HTML, CSS e JavaScript)
- **Tags do Repositório (Topics):** `agrinho`, `sustentabilidade`, `html5`, `css3`, `javascript`

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido com a intenção de conscientizar estudantes e a comunidade agrícola sobre a relevância de se produzir alimentos mantendo a harmonia ecológica. Unindo o tema ao contexto de uma Copa de Futebol, o site oferece uma experiência lúdica e interativa de pênaltis, onde cada acerto promove uma ação sustentável no campo.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de formulários, seções e acessibilidade.
- **CSS3:** Estilização com design responsivo (adaptado para smartphones e desktops) e variáveis para suporte a Modo Escuro.
- **JavaScript (Vanilla JS):** Manipulação de DOM para feedback interativo, controle de placar do jogo e alternador dinâmico de temas.

## 🚀 Como Executar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente em seu navegador web.
